Level 1: Basics (Start here)

Focus: variables, loops, conditions

1. Print numbers from 1 to 50.
2. Print even numbers between 1–100.
3. Check if a number is positive, negative, or zero.
4. Find the largest of 3 numbers.
5. Reverse a number (e.g., 123 → 321).
6. Check if a number is a palindrome (121 → yes).
7. Print multiplication table of a number.
8. Count digits in a number.
9. Sum of digits of a number (e.g., 123 → 6).
10. Check if a number is prime.
